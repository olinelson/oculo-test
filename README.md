## Preview live

This project can be viewed at [oculo-test.vercel.app](https://oculo-test.vercel.app/)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

To run the tests:
 ```bash
npm run test
# or
yarn test
# or
pnpm test
```
